# PM Skills Playground

这是一个基于《产品经理进阶学习方案：基于 AI 驱动的技能矩阵 (PM Skills Marketplace)》整理的静态单页网站，目标是：

- 帮助新手快速理解产品经理核心能力模块
- 提供可直接复制到项目中的 AI 提示词模板
- 作为后续扩展成团队知识库或文档站的基础版本

## 如何打开

直接在浏览器中打开 `index.html` 即可。

## 文件说明

- `index.html`：页面结构
- `styles.css`：视觉样式和响应式布局
- `script.js`：模块切换、模板搜索、复制按钮等交互

## 当前网站包含

- 4 阶段学习路线
- 6 个核心 PM 模块说明
- 8 个可直接复制的项目模板
- 新手避坑提醒

## 适合继续扩展的方向

- 增加更多场景模板，例如需求评审、竞品分析、路线图拆解
- 接入 Markdown 或 CMS，把内容改成可持续维护的资料库
- 增加英文版或团队内部版本
- 部署到 GitHub Pages、Netlify 或 Vercel
