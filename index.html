        创作方法

        短视频不是靠碰运气，而是靠一套能重复的内容系统。

        最简单的理解方式就是 5 步：先定人群，再做选题，再写开头和脚本，然后发布，最后用数据复盘。

        01

        定位

        先讲清拍给谁看、你讲什么、别人为什么要关注你。

        • 目标人群
        • 内容主线
        • 人设与风格
        02

        选题

        不要每天临时想题，先积累选题库，再挑最值得拍的。

        • 误区型
        • 清单型
        • 反差型
        03

        脚本

        一条内容至少要有开头钩子、核心观点和结尾动作。

        • 前 3 秒抓人
        • 主体讲清一件事
        • 结尾引导互动
        04

        复盘

        看 3 秒留存、完播率、互动和涨粉，决定下一条怎么改。

        • 开头留存
        • 中段掉人
        • 转粉与互动

        爆款公式

        最常见的 4 种短视频切入法

        误区型：很多人以为 X，其实真正的问题是 Y。

        清单型：教你 3 个方法、5 个步骤、7 个避坑点。

        反差型：你以为这样更好，结果恰好相反。

        案例型：拆一个真实案例，让观点更具体。

        内容日历

        一个新账号一周内容节奏

        周一：观点建立 周二：误区拆解 周三：案例分析 周四：工具方法 周五：清单总结 周末：复盘与互动

        爆款标题库

        先写 10 个标题，再选最能打的 3 个

        为什么你做了很多内容，却还是没人看 新手做短视频，最容易踩的 3 个坑 别再这样拍了，难怪完播率一直上不去 我用 1 个方法，把短视频选题效率提高了 3 倍 很多人都在学爆款，其实第一步就错了 一个标题公式，帮你把普通内容变得更想点开 做账号之前，一定先想清楚这 5 个问题 同一个主题，为什么别人发就比你更容易爆

        最简单的标题思路：问题感、结果感、数字感、反差感。

        抖音

        更吃开头冲击、情绪反差和快节奏

        适合做强钩子、强情绪、强反差的内容，不适合慢热铺垫。

        适合：误区反转、观点输出、前后对比、结果展示。

        开头：前 3 秒先抛结论、冲突或结果，不要先讲背景。

        节奏：短句、快切、信息密度高，一条只讲一个重点。

        互动：评论区适合做“你属于哪种”“你踩过这个坑吗”这类二选一互动。

        误区：把 B 站式铺垫直接搬过来，开头太慢容易掉人。

        小红书

        更吃经验感、搜索感和可保存

        适合做“有用、可存、可照着做”的内容，比纯情绪表达更容易沉淀。

        适合：清单、步骤、避坑、经验总结、工具推荐。

        开头：先说用户能得到什么，比如“3 个方法”“新手避坑”。

        节奏：可以没那么快，但信息一定要清楚，封面和标题要像答案。

        互动：更适合引导收藏、点赞、留言“要不要继续更这一系列”。

        误区：只学抖音做情绪冲击，却没有实际信息密度，用户不会收藏。

        B 站

        更吃完整表达、信息量和系列化

        适合讲清楚一件事，用户愿意花更长时间看完，也更适合做系列内容。

        适合：案例拆解、方法讲解、深度观点、知识系列。

        开头：可以先交代背景和问题，但要尽快告诉用户这条视频会讲什么。

        节奏：允许更完整的表达，重点是层次清楚，不一定要特别短。

        互动：适合引导弹幕、评论讨论、追更系列和收藏复看。

        误区：把抖音式极短节奏照搬过来，反而会显得浅、信息不够。

        视频号

        更吃信任感、熟人关系和转化闭环

        适合专业分享、行业观点、真人表达和私域承接，比单纯追热点更看重信任。

        适合：专业观点、个人品牌、行业观察、服务型内容。

        开头：要先点明人群和问题，让用户觉得“这条内容就是写给我的”。

        节奏：可以更稳，但要更真诚、更像真人在说话，不要太像流水线脚本。

        互动：适合引导私信、关注、转发给朋友或进入微信生态继续了解。

        误区：只追平台流量感，不做信任承接，转化链路会很弱。

        同一主题怎么改

        抖音：先讲最反常识的一句，再快速给结论。

        小红书：先讲“你能得到什么”，再做步骤和避坑。

        B 站：先交代问题背景，再完整拆清楚逻辑和案例。

        视频号：先点中人群和痛点，再自然承接信任和转化。

        最简单的理解

        抖音看冲击 小红书看价值 B 站看深度 视频号看信任

        标题模板

        不同平台的爆款标题，不能一套文案到处发。

        抖音标题

        更吃反差、结果和情绪

        别再【错误动作】了，难怪你一直【差结果】

        我用【1 个方法 / 3 步】，把【结果】提高了【具体数字】

        为什么你【做了某事】，却还是【没结果】

        小红书标题

        更吃价值、清单和可保存

        【人群】必看：关于【主题】最容易踩的【3】个坑

        一篇讲清【主题】：从入门到能直接上手

        B站标题

        更吃问题、深度和系列化

        【主题】到底怎么做？我用【案例 / 拆解】讲清楚

        为什么很多人学了【主题】还是做不好

        从 0 到 1 讲透【主题】：原理、方法、案例一次说清

        【系列第 1 期】先把【主题】最核心的 3 件事讲明白

        视频号标题

        更吃人群命中、观点和信任

        给【某类人】一个提醒：做【主题】别先急着【动作】

        如果你也在做【主题】，这 3 个问题最好先想清楚

        这是我最近对【行业 / 主题】最真实的一个判断

        很多人问我【问题】，我统一讲讲我的做法

        互动话术

        评论区不是只用来求赞，更是放大讨论和承接转化的地方。

        下面把四个平台更自然的互动方式和转化句式拆开了，尽量避免一上来就太硬。

        抖音评论区

        适合快互动、轻转化

        互动：你更像哪一种？留言 1 或 2 我看下大家比例。

        互动：你踩过这个坑吗？我想看下有多少人中招。

        转化：如果你想看下一条，我继续拆【下一个问题】。

        转化：主页我放了同系列内容，想系统看可以接着翻。

        小红书评论区

        适合收藏、补充和软承接

        互动：如果你也在做【主题】，可以留言“继续”，我再更下一篇。

        互动：你最想补哪一部分？我按评论区需求继续写。

        转化：这篇适合反复看，先存一下，后面做的时候直接翻。

        转化：我把同系列都放在主页合集里了，想系统看可以直接点进去。

        B站评论区

        适合讨论、追更和系列沉淀

        互动：你更想看我继续拆案例，还是继续讲方法？评论区告诉我。

        互动：如果你有不同观点，也欢迎在评论区补充，我们一起完善这个话题。

        转化：这一期先讲到这里，下一期我继续拆【主题】里最难的一部分。

        转化：如果你想系统看，这个系列我会持续更新在同一个合集里。

        视频号评论区

        适合建立信任和自然承接

        互动：如果你也正卡在这个问题，留言告诉我你的场景，我下次按真实问题继续讲。

        互动：这条内容更像是给正在做【主题】的人一个提醒，你现在在哪个阶段？

        转化：后面我会继续分享更具体的做法，关注一下，免得错过。

        转化：如果你想看更完整的一套内容，我会继续放在后面的系列里慢慢讲。

        3 秒钩子

        用户不是没耐心，而是你没有在前 3 秒告诉他为什么要继续看。

        下面这些开头句式适合拿来直接改，重点是先抛问题、结果、反差或身份认同。

        问题型

        为什么你做了这么久，还是没有结果？

        你是不是也卡在这个问题上很久了？

        很多人一开始就做错了第一步。

        反差型

        很多人以为这样更好，其实恰好相反。

        你以为是内容不行，其实问题根本不在这。

        越努力越没结果，往往就是因为这个。

        结果型

        我只改了一个点，结果直接差很多。

        这个方法帮我把效率提高了 3 倍。

        如果你也想更快拿到结果，先看这一步。

        身份型

        如果你是新手，这条一定先看。

        做过这件事的人，一定知道这个坑有多大。

        给正在做【某件事】的人一个提醒。

        清单型

        今天只讲 3 个最关键的点。

        新手最容易忽略的 3 个细节，我一次讲清。

        这份清单你可以直接照着做。

        案例型

        我拿一个真实案例，带你看清楚为什么会这样。

        这个例子一出来，你就知道问题在哪了。

        别空讲原理，我们直接看真实案例。

        直播与私域

        直播预告和私域承接的关键，不是喊用户来，而是先说清楚他来了能得到什么。

        下面把预告、开播和收尾的常用话术拆开，尽量用更自然的表达，不做硬推。

        直播预告

        先讲时间，再讲价值

        今晚 8 点直播，我会把【主题】从头到尾讲清楚。

        如果你也卡在【问题】，今晚来直播间，我会直接拆给你看。

        这次直播我会讲 3 件最核心的事，适合新手直接来补课。

        不是泛聊，我会带着案例讲，来的人可以直接拿走方法。

        开播话术

        先拉近关系，再给预期

        刚进来的朋友先别急着走，今天这场我主要讲【主题】。

        如果你正在做【场景】，今天这场会比较有用，我先把框架讲清楚。

        后面我会用一个真实案例带大家拆，所以先留几分钟听完开头。

        有问题可以直接打在评论区，我会边讲边看。

        私域承接

        更自然地引导下一步

        如果你想继续系统看，我后面会把这一系列继续整理出来。

        想看更完整版本的，可以先关注，后面的内容我会接着更。

        如果你更想看实操版，我后面会继续放具体案例和模板。

        收尾转化

        别硬推，先做轻承接

        今天先讲到这里，想看下一部分的可以留言，我按需求继续讲。

        如果这场对你有帮助，先点个关注，后面我继续把这一套讲完整。

        你最想让我下一场拆哪一块，评论区告诉我，我按真实问题来安排。

        先别急着走，回头把今天的几个关键点记一下，后面你会用得上。

        账号简介

        一个好的账号简介，不是介绍自己，而是让别人 3 秒知道“为什么要关注你”。

        最稳的结构通常是：我是谁 + 我帮谁 + 我持续分享什么 + 为什么值得关注。

        定位型简介

        我是【身份 / 角色】，专门分享【主题】。

        这里持续更新【内容方向】，适合【目标人群】。

        关注我，带你少走弯路。

        结果型简介

        帮你更快搞定【目标结果】。

        这里分享【方法 / 工具 / 案例】。

        适合正在做【场景】的人。

        陪伴型简介

        我也在持续做【主题】，这里记录真实过程。

        会分享踩坑、方法和复盘。

        适合想一起成长的人。

        专业型简介

        【职业 / 从业年限 / 领域】。

        长期分享【专业方向】的实操经验。

        希望把复杂问题讲得更简单。

        商务合作

        商业合作和报价话术的关键，不是报一个数字，而是先讲清合作范围、交付内容和边界。

        最自然的表达方式通常是：先确认需求，再给合作方式，最后给报价和补充说明。

        初次回复

        感谢联系，我先了解一下这次合作的产品、目标和发布时间。

        如果方便的话，也可以先发下合作需求和预期形式，我这边评估后回复你。

        我通常会先看是否匹配账号内容，再决定怎么合作更自然。

        报价表达

        这次如果按【合作形式】来做，报价区间会在【数字】左右。

        这个报价通常包含【脚本 / 拍摄 / 发布 / 修改次数】这些部分。

        如果交付范围有变化，我也可以再按实际内容调整。

        边界说明

        我更适合合作【内容类型】，不太接【不匹配类型】。

        如果需要强硬口播或与账号调性差距太大，我一般不会硬接。

        我会优先考虑和现有内容风格一致、用户也更容易接受的合作方式。

        婉拒话术

        感谢邀请，这次合作方向和我当前内容规划不太匹配，我就先不硬接了。

        我更希望每次合作都和账号调性保持一致，所以这次先不接，感谢理解。

        如果后面有更贴近【主题】的合作，也欢迎再联系我。
